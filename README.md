# vuejs-tab-tracker
Full-Stack Guitar Tab tracking web app using Vue.js &amp; Express.js

## Requirements 
- node.js v8+

## Setup
- Install server dependencies (Express) & start server side
```$xslt
cd server
npm i
npm start
```
- Open a new terminal window & install client dependencies (Vue.js) & run client
```$xslt
cd client
npm i
npm start
```
